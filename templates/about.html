WHAT IS PULL.LY?

Pull.ly is a free to use application that allows github users to sign into their account and view all their pull requests in one simple and easy to navigate centralized viewing platform. Users may view, comment, and review pull requests at their leisure.

Users may manage pull requests, using the platform to review, post comments and moderate the comments on pull requests.

The only way to learn how to use it is to try it out. So, let's get started!
